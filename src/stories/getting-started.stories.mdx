## Getting Started

### Installation

To get started using the project, follow the steps below:

- Clone the [Elizabeth](https://github.com/santiago-enciso-q2/elizabeth) repo from GitHub

```
git clone git@github.com:santiago-enciso-q2/elizabeth.git
```

- Install npm dependencies

```bash
npm i
```

- Start project

```
npm start
```

- Access project in [http://localhost:3000](http://localhost:3000)
